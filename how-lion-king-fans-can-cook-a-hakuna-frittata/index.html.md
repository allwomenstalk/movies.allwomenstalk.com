# How Lion King Fans Can Cook a Hakuna Frittata ...

If you're a huge fan of The Lion King, then you should celebrate the film today. How? Well, you can simply put on the movie while eating a delicious meal that goes along with it.

The video down below will teach you how to create a Hakuna Frittata. You're going to need millet, pork, eggs, fried okra, basil, celery leaves, and BBQ sauce. It's a super easy recipe to follow, which means that anyone can create it.
