# Movienizer.com Announces the Release of Movienizer 1.6, Free Software for Movie Fans to Catalogue, Track and Research Movies

(PRWEB) April 24, 2008

(PRWEB) Movienizer is an advanced movie organizer that keeps all information about movies and videos in one place. It can use powerful online databases to retrieve information about movies and personalities. With movies, the following information is available: description (synopsis), front cover, snapshots from the movie, the complete cast, personal notes etc. A location and disc number may be assigned to each movie, which makes searching for an actual DVD, HD-DVD, Blu Ray, and other media a cakewalk of a job.

Movienizer keeps track of loaned media. It can also easily list all seen (or not seen) movies in the collection. Its advanced search engine helps finding a movie or a number of movies by any criterion, including genre, MPAA rating etc. The program is capable of generating printer-friendly reports, using either the whole collection, or just search results.

The collection is fully customizable, which means that any field can be edited manually. The user can preview movies and take snapshots with the help of the built-in player. External media players are also supported, so the user can start playing any movie directly from Movienizer.

One of the most valuable features of Movienizer is the link between movies and personalities. If the user likes some actor and wants to learn about other movies with the actor, it is sufficient to click the actor's name and hit "Download". The complete filmography will be downloaded, along with photos, biography, other details. To find out about a particular movie from the filmography, the user can simply click the movie and hit "Download". After getting acquainted with details of the movie, the user may add it to his or her wish list.

In this way, it is easy to explore the world of movies by navigating from a movie to a person, and then to other movies. The whole collection gets a new dimension, from now on it is more than just a number of videos.

For more information, please visit [movienizer.com](https://www.movienizer.com/)
